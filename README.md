# api backend mongo

## Description
  Series backend on mongo


## <a name="important"></a> Before Getting Started

### Important


### Start install dependencies

```sh
yarn
```


### copy environment

```sh
cp .env.example .env
```


### Run the application


```sh
yarn dev
```

## <a name="env"></a> Env Variables

- `HOST` Host name
- `PORT` port express
- `BASE_API` base url to /api
- `DATABASE_URI` parameters to mongo string connection
